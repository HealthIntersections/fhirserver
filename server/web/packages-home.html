How to add packages to this package server:

  1. Create an RSS feed, using the template at http://hl7.org/fhir/package-feed.xml, and then register the package feed here in https://github.com/FHIR/ig-registry/blob/master/fhir-ig-list.json by making a GitHub PR. The Package server will check the RSS feed for new packages every hour
  2. Ask the FHIR product director for a token that allows you to submit packages to the server directly, using the API
